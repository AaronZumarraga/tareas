# Components
Componentes reutilizables y pequeños: piezas UI atómicas o moleculares.

## Ejemplos
- Botones (`BaseButton.vue`)
- Entradas (`TextField.vue`)
- Tarjetas (`Card.vue`)
- Tablas (`DataTable.vue`)
- Modales (`Modal.vue`)
- Loaders (`Spinner.vue`)

## Convenciones
- Prefijo opcional `Base` para componentes neutros
- Sin lógica de negocio pesada
- Props claras y tipadas
- Emitir eventos estandarizados (`update:valor`, `submit`, etc.)
